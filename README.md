Secure Hotel Management System
Overview :
The Secure Hotel Management System is a Python-based application designed to manage hotel operations with enhanced security.
It focuses on protecting sensitive user data using encryption techniques while handling bookings and customer records efficiently.

Features
- Room Booking Management
- Customer Data Handling
- Secure Login System (DBMS-based authentication)
- Data Encryption using AES Algorithm
- Reservation Tracking
- Basic Billing System
- Security Features

Technologies Used
- Programming Language: Python
- Database (DBMS): MySQL / SQLite
- Security: AES Encryption

Setup database:
- Create database in MySQL / SQLite
- Import required tables

Run the application: python main.py

How It Works: 
- User logs in through a secure authentication system
- Credentials are verified using DBMS
- Sensitive data is encrypted using AES before storage
- System manages hotel operations securely
